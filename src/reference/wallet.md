# Wallet类

# 阿是多少
## Wallet.generate()
创建新的钱包

返回值:
| 参数 | 类型 | 说明 |
| :-----| ----: | :----: |
| secret | String | 井通钱包私钥 |
| address | String | 井通钱包地址 |

## Wallet.fromSecret(secret)
基于密钥创建钱包

字段:
| 参数 | 类型 | 说明 |
| :-----| ----: | :----: |
| secret | String | 井通钱包私钥 |

返回值:
| 参数 | 类型 | 说明 |
| :-----| ----: | :----: |
| secret | String | 井通钱包私钥 |
| address | String | 井通钱包地址 |
